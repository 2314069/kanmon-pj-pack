# 感門之盟について意味を再解釈するロール

- 学衆さん、師範代ら参加者へのお祝いの場
- 相手を言祝ぐことが一番大事(ほめる前提で人に言葉をかける)

# 感門之盟のテーマについて誰よりも自由に語れる

- 感門之盟のテーマはニューワードであり、キーワード
- どんなものからでもテーマに繋げて語れるように
- 感門之盟のひとつひとつのコーナーを感門テーマに絡められるようにする

# 場を引き取るのが司会

- 何を引き取って何を広げるのか、一番最初に相互編集できるのが司会者
- 手前の言葉、ふるまい、さしかかりを自分なりの言葉で言いかえる
- 大事なことを繰りかえす、文脈を再生する
- その場をどうしたいのかを常に判断する
- コーナー司会や他のロールに積極的にディレクションを入れていく

# 場の流れを作るのが司会

- 場のメリハリ、緩急、濃淡を作るのも司会者の役割
- ピークアウトを作る
- 場の格を保つように手前を受ける
- 次に振ることも大事な役割
- のしつきで人を紹介する

# 依り代となるのが司会

- 宇宙的礼節を持つ
- 場の様子を言語化していく
- 自分なりの憧れや肖りを持って司会を演じる
-　その場に起こったこと、放たれたことを何よりも大事にする
- 何かひとつかふたつ半年間響き続けるメッセージを用意すること

# その場のさしかかりを一番に察知するのが司会

-  どんなさしかかりにも気づくようセンサーを研ぎ澄ませる
- 小さなさしかかりを拡張する、意味あるものにする

# 応じるホドをみせるのが司会

 - 基本的にメッセージが出るのは学衆や師範代、師範から。そのメッセージを受けて何を返すか
 - 応じるホドに編集を効かす
 - 三人称的に語る場合と相手に話しかける場合のきりかえ

# 司会の用意と卒意

- コーナーの説明や人の説明など原稿を用意しておく
- 展開の言葉、接続の言葉、受けの言葉、賞賛の言葉を用意しておく
- 話始めの言葉、締めの言葉までレパートリーをそろえておく
- 原稿を用意した上でその場に出たものを拾っていく

# 司会の言葉使い

- 感門之盟に相応しい言葉使いについてイメージを固める
- 端的に、短く要約するトレーニングをする。意味や価値を抽出するトレーニングをする。
- 独自の尺度でスコアリングする。
- 学衆にとっても伝わる言葉に言いかえる
	- イシス用語は事前に調べて自分なりに語れるようにする

# 司会のフォーマット

- ステージに立つときには、演出や手立てが必要
- 演出のためのフォーマットを用意すること

# その他

- うなずく前に言いかえる
- 二手三手進めて引き出せ
- 100のif-thenを考えておく
- どんなものを届けられるか考えるときには、その人の奥を見るといいよ
- どんなささいなことでもよほど大きなこととして受け取ってほしい
- 惜しまないこと
- この司会者だからこそという安心感
- 要所要所のコメント力が求められる
- 感を動かしながらコメントをしていく
- ふるまいとしつらえが一緒になった存在が司会ロール
- 司会者のその人らしさを大事にする
	- キャラクターやその人の立ち振る舞いや姿ごと記憶に残っていく
	- その人のカマエが全身見える、その印象が残っていく
- 司会者は依り代でしかないという意識
	- 何かひとつかふたつ半年間響き続けるメッセージを用意すること

# 司会準備の「読前・読中・読後」フレームワーク

> 88-89感門の司会経験から得られた準備方法。感門テーマから司会のロールやプログラムの流れを読書とインタースコアさせて考える。

## 基本コンセプト

**感門之盟は壮大な共読体験である**

## 読前（オープニングから卒門式まで）

- タイトルや装丁や帯や目次から本の内容のイメージを広げている段階
- 司会のロールは、**本を開く前の読者（参加者）の様子や感情を言葉にする**こと

## 読中（卒門式から続くコーナー）

守の用法１〜４に当てはめる：

| コーナー | 用法 | 司会の受けコメント |
|---------|------|------------------|
| 感門表授与 | 用法１ | わける／あつめる |
| 先達文庫／師範頌授与 | 用法２ | つなぐ／かさねる |
| （学衆賛頌） | - | 司会ロール的に休憩扱い |
| 出世魚教室名発表／ISIS Now | 用法３ | しくむ／みたてる |
| 学長対話 | 用法４ | きめる／つたえる |

### 準備のポイント

- どの型を使ってコメントを組み立てるか、ターゲットをどこに置いておくか、おおよそ見当をつけておく

## 読後（林頭コールから）

- 締めコメントでは、**読前の仮説を振り返り**、ブックウェアを自分なりに言い換え、**語り手を突出させる**
- 学長対談を聞きながら、締めの言葉を準備

## このフレームワークの効果

- 事前アンケートへの回答、インタビュー、その後の準備を振り返ると有効だった
- **何を話すかを用意しておくのではなく、どのようなカマエで臨むのか**を固めておくことが重要
- プログラムの流れを自分なりにどう意味づけるかを事前に決めておく

# 司会インタビュー（麻人システム）

> 校長のリハディレクションが欠けた分を埋めるための施策として画期的だった方法。

## 内容

1. 過去の司会担当の感門後振り返りを集められる限り集める
2. コーナー司会もふくめてMC全員を集めミーティングを行う
3. 個別にインタビューをして、その人のダント・数寄・編集への思いを引き出す
4. インタビュー内容を記事にする

## 効果

- 自分の司会モデルを仮説できた
- 司会モデルを具体化させる方向で準備を進められた
- オラリティの方法化があまり行われてこなかったこれまでの感門司会準備と比べても画期的

## 継続の推奨

今後、司会を担当することになった人は、この「麻人システム」を体験できることを楽しみにしていただくとよい。
