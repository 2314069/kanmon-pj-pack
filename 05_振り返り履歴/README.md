# プロジェクト履歴

> 過去の感門之盟プロジェクトのアーカイブです。

## フォルダ構成

```
05_プロジェクト履歴/
├── 88期/
│   ├── 次第/
│   ├── 参加者名簿/
│   ├── 感門団ロール表/
│   ├── 座席表/
│   ├── 議事録/
│   └── 振り返り.md
├── 89期/
│   └── ...
└── ...
```

## 各期のフォルダに入れるもの

- **次第**: 確定版の次第表
- **参加者名簿**: 最終版の参加者リスト
- **感門団ロール表**: 当日使用したロール表
- **座席表**: 当日使用した座席表
- **議事録**: 各MTGの議事録
- **振り返り.md**: PJ完了後の振り返りメモ

## 振り返りテンプレート

```markdown
- 参加人数：

-

-

-
```

## 注意事項

- 個人情報を含むファイルの取り扱いに注意
- 振り返りは次期の準備に活用できるよう具体的に記載
