# 感門之盟 PJパック

> 感門之盟の準備・運営に必要な情報をまとめたナレッジベースです。

---

## 🎯 このフォルダの目的

感門之盟の準備を効率的に進めるために、以下を整備しています：
- **ロール・ルール・ツール**の定義
- **タスク・ダンドリ**の一覧
- **テンプレート**の提供
- **過去のナレッジ**の蓄積

---

## 📂 フォルダ構成

| フォルダ | 内容 | いつ使う？ |
|---------|------|-----------|
| [01_ルル](./01_ルル/) | ロール・ルール・ツール | PJ開始時に確認 |
| [02_ダンドリ](./02_ダンドリ/) | タスク・スケジュール | 準備期間中に参照 |
| [03_テンプレート](./03_テンプレート/) | 各種ひな形 | ドキュメント作成時 |
| [04_ナレッジ](./04_ナレッジ/) | 過去の知見・ノウハウ | 困ったときに参照 |
| [05_プロジェクト履歴](./05_プロジェクト履歴/) | 過去期の実績 | 振り返り・参考に |

---

## 🚀 クイックスタート

### 統括ロールの方へ
1. [ロール.md](./01_ルル/ロール.md) でロール構成を確認
2. [全体ダンドリ](./02_ダンドリ/00_全体ダンドリ.md) でタスク全体像を把握
3. サブチームの立ち上げ・メンバーアサイン

### サブチームリーダーの方へ
1. 担当チームのダンドリファイルを確認
   - [感門団](./02_ダンドリ/01_感門団.md)
   - [感門エディスト](./02_ダンドリ/02_感門エディスト.md)
   - [MCラウンジ](./02_ダンドリ/03_MCラウンジ.md)
   - など
2. [テンプレート](./03_テンプレート/) からキックオフ議事録のひな形を取得
3. [ナレッジ](./04_ナレッジ/) で過去の知見を参照

## 📁 詳細フォルダ説明

### 01_ルル
感門準備の「3R」（ロール・ルール・ロール）を定義。
- `ロール.md` - 必要なロールと責任範囲
- `ルール.md` - ドキュメント管理などのルール
- `ツール.md` - 当日使用するツールリスト

### 02_ダンドリ
全体およびサブチームごとのダンドリ。
- `00_全体ダンドリ.md` - 全体タスクとスケジュール
- `01_感門団.md` - 感門団チームのダンドリ
- `02_感門エディスト.md` - エディストチームのダンドリ
- `03_MCラウンジ.md` - MCラウンジのダンドリ
- `04_次第作り.md` - 次第作成のダンドリ
- `05_冊子.md` - 冊子（Editorship）のダンドリ
- `06_テーマ決め.md` - テーマ決定プロセス
- `07_アフ感.md` - アフターパーティーのダンドリ

### 03_テンプレート
各種ドキュメントのひな形。
- `ドキュメント/` - Excel・PowerPointテンプレート
- `議事録ひな形/` - MTG議事録のテンプレート

### 04_ナレッジ
過去期からの知見・ノウハウ。
- `司会・MC/` - 司会ロールに関するナレッジ
- `運営ノウハウ/` - 運営全般のノウハウ
- `振り返り・課題/` - 改善点・今後の課題

### 05_プロジェクト履歴
過去期のプロジェクトアーカイブ。

## 📝 更新ルール

[ルール.md](./01_ルル/ルール.md) を参照してください。

### 基本ルール
- BOXにアップしたファイルは必ずBOX内で編集
- ローカル修正時は必ずファイル名を変更してアップ
- 古いファイルは「履歴」フォルダに格納

## 👤 統括ロールの方へ

PJ中必要に応じて本フォルダを更新してください。
特に以下の更新をお願いします：
- 新しいナレッジが得られたら [ナレッジ](./04_ナレッジ/) に追加
- 改善点があれば [今後の課題](./04_ナレッジ/振り返り・課題/今後の課題.md) に追記
- PJ完了後は [プロジェクト履歴](./05_プロジェクト履歴/) にアーカイブ
