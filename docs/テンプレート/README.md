> 感門準備で使用する各種テンプレートです。

---

## ドキュメントテンプレート

Excel・PowerPointテンプレートの格納場所です。

| ファイル名 | 用途 | 使用タイミング |
|-----------|------|--------------|
| 座席表ひな型.pptx | 当日の座席配置図 | 参加者FIX後 |
| 参加者名簿.xlsx | 参加者リスト管理 | 準備期間中随時更新 |
| 次第.xlsx | 感門当日の次第表 | 次第作り時 |
| ロール表.xlsx | 感門団のロール割り振り | 感門団ロール決定時 |

{% hint style="info" %}
**テンプレートの場所**

実際のテンプレートファイルは[BOX](https://app.box.com/folder/353479667744)で管理しています。
{% endhint %}
