## 全体ダンドリ

1. **感門タイトル決定**
    - 与件の整理

2. **感門団ラウンジ作成**

3. **メンバー登録**
    - 関係講座の師範、番匠、花目付、学匠、学林局
    - JUST記者との連携者

4. **ラウンジ開設の投稿**
    - ラウンジの位置付け
    - ラウンジメンバー
    - 目的の共有
    - ルール（Box案内、ドキュメント更新など）の共有

5. **サブチーム作成**（統括＋講座PL＋チームメンバー）
    - 感門団チーム
    - エディストチーム
    - 冊子チーム
    - アフ感チーム

6. **ロール決定**
    - 講座ごとの感門団PL決定
    - サブチームごとのメンバー募集
    - 総合司会

7. **サブチームごとにダンドリ進捗**

8. **事前準備**

9. **事前リハ**

10. **当日**

11. **振り返り**
