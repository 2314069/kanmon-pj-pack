## フェーズ1: 立ち上げ

### 統括・PLの決定

- [ ] 統括ロールを決定する
- [ ] 各講座から1人～2人の感門団PLを決定する
  - 感門団PLに講座と感門団を繋いでもらう

### タスク洗い出し

- [ ] 当期外から何人募集するかの決定
- [ ] 次第の確認
- [ ] 準備日に用意するもの確認
- [ ] 必要情報の洗い出しと取得方法の検討

### キックオフMTG

- [ ] ロールの確認
- [ ] タスクの認識合わせ
- [ ] スケジュールの認識合わせ

## フェーズ2: メンバー募集・ロール決定

### 感門団の募集

- [ ] 当期感門団募集
- [ ] 当期外感門団募集

> **期外感門団の意義**: 当期師範の負担軽減、継承、学校への関わり方の多様化

### 感門団ロール表の作成

- [ ] [感門団ロール表](https://eel-jp.box.com/s/ze2exryb61nwflf1ripzrediqyf15ld1)の作成

**ロール設定のポイント:**
- 当期・当期外とも、感門団経験や裏方、黒膜衆などの経験がある方をキーマンに
- 残りのロールを次第と登壇を眺めながら振っていく
- **「受付待機ロール」を必ず決めておく**（忘れがち！）

### ロール一覧チェック

| ロール | 担当者 | 備考 |
|-------|-------|------|
| 統括 | | |
| 受付 | | |
| **受付待機** | | 途中参加・退席者対応 |
| タイムキーパー | | |
| マイク出し | | |
| カウンター | | 2名連携がポイント |
| クローク | | |
| 軽食準備 | | |
| 席移動・誘導係 | | |

### 読み合わせMTG

- [ ] 感門団ロールの読み合わせ実施
- [ ] 各ロールのルルを事前に共有（当日朝のmtgだけでは弱い）

## フェーズ3: ドキュメント作成

### 座席表の作成

- [ ] 参加者FIX後、[座席表](https://eel-jp.box.com/s/ze2exryb61nwflf1ripzrediqyf15ld1)を作成

**注意点:**
- 本楼の照明は暗めなので文字サイズに注意
- ガレージ側にも掲示が必要
- 移動する教室担当の師範には拡大版を印刷

### 受付名簿の作成

- [ ] [受付リスト](https://eel-jp.box.com/s/l0mxide9emfbpifoyffinaxpth9b3ocg)を作成

**形式の選択:**

| 形式 | メリット | 向いているケース |
|-----|---------|----------------|
| 50音順 | 初めての人でも探しやすい | 期外メンバーが担当、複数講座合同 |
| 教室ごと | 当期メンバーには馴染みやすい | 当期メンバーのみで担当 |

> 結構直前で修正があるため対応は当期外感門団で引き受けた方がよい

## フェーズ4: 事前準備

### 名札準備

- [ ] 名札作成（印刷）
- [ ] ストラップの色ごと、教室ごとにまとめる
- [ ] **白紙の名札用用紙を準備**（必須！）

### 物品準備

- [ ] ペットボトルのラベル貼り
- [ ] お菓子準備

### 講座企画関連（必要に応じて）

- [ ] P1の景品
- [ ] インターブッキングのブックカバー
- [ ] 引き換え札

### 案内掲示物

- [ ] 座席表（躙り口側、ガレージ側）
- [ ] 男子トイレへの誘導表示
- [ ] 立ち入り禁止エリアの表示
- [ ] クローク案内

> **改善提案**: パウチして保存しておくと再利用できる

## フェーズ6: 当日

### 朝会（全員集合）

> 複数講座同一開催の場合は、全体の顔合わせが特に重要

**確認事項:**
- [ ] 次第の最終確認
- [ ] ロール分担の確認
- [ ] 名札・インターブッキング本の所在確認
- [ ] 連絡手段（LINE等）の確認

→ 詳細は [当日ダンドリ](感門団当日.md) を参照

## よくある落とし穴

1. **受付待機ロールの漏れ**
   - オープン後も受付に一人はいる状態にする
   - 途中参加・退席者対応に必要

2. **当期師範のダブルロール**
   - 登壇を控えている中でのダブルロールは難しい
   - 期外感門団に引き取った方が良い場合も

3. **白紙名札の準備忘れ**
   - 名簿に反映できていなかった方用に必須

4. **事前のルル共有不足**
   - 当日朝のmtgだけでは弱い
   - 事前にロールの動き方を共有しておく
