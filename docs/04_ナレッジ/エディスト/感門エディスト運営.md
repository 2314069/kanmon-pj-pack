# 感門エディスト運営

> 感門エディストチームの立ち上げから運営までのノウハウをまとめたドキュメントです。

---

## チーム立ち上げの流れ（実績例）

```
7月14日　ロール発表
7月15日　感門講座ごとのリーダー募集
7月26日　ラウンジ上でプレ旗揚げ／要点の整理
8月 4日　キックオフミーティング
8月 8日　記事公開のスケジュール共有
8月31日　第2回ミーティング（外部記者連携）
```

### 成功のポイント

- **1か月前にキックオフミーティングができたことが大きい**
- 講座連携型の感門エディストチームとして新しく立ち上がった

---

## 記事の分類

### 3つのフェーズ

| フェーズ | 目的 | 例 |
|---------|------|---|
| 事前記事 | 「面白いことが始まりそう」という空気感を伝える | タイトル決定、インターブッキング、司会紹介 |
| 当日記事 | リアルタイムで感門の熱を伝える | 各コーナーレポート、ダスト記事 |
| 事後記事 | 振り返りと記録 | まとめ記事、P-1記事 |
