[:material-link: ホットペッパー](https://www.hotpepper.jp/strJ000980615/){ .md-button }

| 項目 | 内容 |
|-----|------|
| 営業時間 | 20時〜22時（21:30LO）15分前から入場可 |
| 電話 | 03-6413-8588 |
| アクセス | 下北沢駅から徒歩５〜６分ほど |
| 費用 | 4500円（税込み） |

近隣が住宅で少し静なエリアの為、入場／退場の時お店の周りでたまって賑やかにならないようにというお願い有り。お店では賑やかでＯＫ。なので、アフアフ希望の方はスムーズに誘導ができればと思います。
