## イキイキ編集を語ること

難しいこと抜きに、まずなによりも司会がイキイキと編集（講座）のおもしろみについて語れること、そして学衆さんのその先の編集道（可能性）を思ってワクワクできることがいちばん大事！

---

## 台本作り

司会ロールもプロセス・エンジニアリング。「準備」という編集プロセスそのものがおもしろい。愉快な表現や掛け合いの言葉を練ること楽しむ（衣装合わせも！）

- MC自身がどんなイシス人なのか、コーナー冒頭などでパパっと自己紹介する必要があるので、この機会にあらためて、司会者自身の編集学校歴（教室名、編集との出会い、守破花の稽古ぶり、校長との関わり、フェチ、数寄、憧れ、得意手、ヘンなところ、などなど）をたどりなおしておく

{% hint style="info" %}
**ヒント**

こういう機会に、自分の編集クロニクル振り返ってみる作業自体が、愉しいと思います！
{% endhint %}

- そうやって集め直したクロニクルの中から、「ここぞ」という情報をつまみだして、感門の場に出す。冒頭の自己紹介では使わなかった情報も、場に応じての引き出しになりうる

## 司会とは

メディア・媒介者・つなぎめ・ハブ・場・パッサージュ・共身体・ナレーター

### 引き出す役割（登壇者の姿をリプレゼントする）

- 自分が何かを主張するというよりも、登壇者や観客の中から情報を引き出して言語化する役割。師範代の指南と基本的には同じ。
- 「相手の中に潜むおもしろポイント」を拾う、敏感な注意のカーソルとしてあること。

### 師範代や師範の「あらわれ」を「あらわし」にする

- エディスト名鑑のくすぐるような書きぶりや、康代学匠・原田学匠が先達文庫授与の際にひとりひとりにかける言葉がモデルになる

## コーナーの受け方

### 観客の無意識を言語化する

- コーナーを聞いている学衆のアタマに浮かんでいる吹き出し（無意識）に輪郭をあたえ、二～三個拾い出して言語化してあげるイメージ。

{% hint style="success" %}
**松岡校長の例**

コーナーの入りはだいたい「なんかいまのヘンだったよね」「彼はおもしろかったよね」と、直前のコーナーに言及することで観客の中の与件（感想）を拾うところから入っていって、そこからなだらかに本題へ入っていく
{% endhint %}

### 語りの切り替え

感門のような複数人を相手する語りは、司会席近辺にいる「あなた」に話しかけるような二人称的語りと、誰に言うでもなく全体に対して伝えるような三人称的語りと、２種類ある

→この切り替えがメリハリを生んでくれる

### 間と表情

「言葉」をあまりいそぎすぎず、「間」や「（自分の）表情」もひとつの情報として活かす。ときには司会がじっと静かに感じ入るようなことも、校長は大事にしていた

## 講座が図・感門テーマは地

破知文の「本・著者・読者」の三角形になぞらえると：

```
      講座
    ／　　＼
テーマ―― 司会
```

- 学衆にとっての一番の関心は講座の内容。感門テーマは背景情報として宙に浮かべておき、いつでもトピックのつなぎ目として使えるようにしておく

- 感門テーマ「遊撃ブックウェア」については、「本」「読書」の連想がちゃんと広がっていれば、それがそのままブックウェアの話にも接続されそう

- 「遊撃」のほうは、「編集は遊びから生まれる」はもちろん、たとえば『編集宣言』冒頭にもあるように、「遊」がコドモやオニが外に遊びに行って帰ってくること、往来や往還を本質としていることなどから、イシスのコースウェアにおける行ったり来たりや塗り重ねとも親和的

---

## 司会は基本「受け」だが、ときに「講座アピール」を攻める！

- 学衆をその先の講座に導くこと、その先の講座のおもしろみを示すことも司会の役割
- 「ここぞ！」というところで、自分の講座体験や、その講座のここがおもしろい！という点を、コーナーに重ねて語るのもよい

---

## 次第構成の抜き打ちトレース＆リバース

- 次第を思い出せるかどうか、ランダムに自分に課して、コーナーの流れを取り出しやすいようにしておく
- 台本（次第）をプリントアウトした紙自体が、よくもわるくもトポスとして機能してしまう
    - →頼りすぎると、「台本片腕密着身体」になってしまう
- 台本を離れても、どんな紙の上でも構成を思い出せるようにしておくとベスト。なんども想起していくうちに、構成やセリフが略図化・イメージ化されていくはず

{% hint style="info" %}
**参考**

『世界のほうがおもしろすぎた』のＰ226に、校長ならではな舞台パフォーマンスの際の記憶方法があかれされています
{% endhint %}

## ペア司会の場合

ペア司会の場合には「どんな二人組モデルなのか」、もっておくとちょっと楽しいかも

- 自分たちは、ボケ・ツッコミなのか、ボケ・ボケなのか、ツッコミ・ツッコミなのか、場面によって流動的なのか…

- もし世にあるコンビにたとえるなら、サンドイッチマンなのか、オードリーなのか、ピンクレディーなのか、ドゥルーズ＝ガタリなのか、石牟礼道子・渡辺京二なのか、おすぎとピーコなのか、ぐりとぐらなのか…笑

{% hint style="info" %}
**ポイント**

明確に特定できるモデルがなくてもいいですが、「こんなコンビ」というざっくりイメージがあると、司会進行もやわらかく展開しやすいかもです。
{% endhint %}

{% hint style="success" %}
**新井・若林ペアの例**

私と若林さんは、「天然コンビ」とか「溢れる二人」とか位置づけられてましたし（[参考記事](https://edist.ne.jp/just/78kanmon_day2mc/)）自分たちでも打ち合わせ段階で「過剰さ」を自覚・意識して、冒頭の掛け合いからあふれ出す語りでかっとばしていくことを心がけました
{% endhint %}
