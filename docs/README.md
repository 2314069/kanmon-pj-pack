# 感門之盟 PJパック

> 感門之盟の準備・運営に必要な情報をまとめたナレッジベースです。

---

## このサイトの目的

感門之盟の準備を効率的に進めるために、以下を整備しています：

- **ロール・ルール・ツール**の定義
- **タスク・ダンドリ**の一覧
- **テンプレート**の提供
- **過去のナレッジ**の蓄積

---

## クイックスタート

### 統括ロールの方へ

1. [ロール定義](ルル/ロール.md) でロール構成を確認
2. [全体ダンドリ](ダンドリ/README.md) でタスク全体像を把握
3. サブチームの立ち上げ・メンバーアサイン

### サブチームリーダーの方へ

1. 担当チームのダンドリファイルを確認
   - [感門団](ダンドリ/感門団.md)
   - [感門エディスト](ダンドリ/感門エディスト.md)
   - [MCラウンジ](ダンドリ/MCラウンジ.md)
   - など
2. [テンプレート](テンプレート/README.md) からキックオフ議事録のひな形を取得
3. [ナレッジ](ナレッジ/README.md) で過去の知見を参照

### 基本ルール

- BOXにアップしたファイルは必ずBOX内で編集
- ローカル修正時は必ずファイル名を変更してアップ
- 古いファイルは「履歴」フォルダに格納
