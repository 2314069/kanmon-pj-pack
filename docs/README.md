# ホーム

> 感門之盟の準備・運営に必要な情報をまとめたナレッジベースです

***

## このサイトの目的

感門之盟の準備を効率的に進めるために、以下を整備しています：

* **ロール・ルール・ツール**の定義
* **タスク・ダンドリ**の一覧
* **テンプレート**の提供
* **過去のナレッジ**の蓄積

***

## クイックスタート

### 統括ロールの方へ

1. [ロール定義](ルル/ロール.md) でロール構成を確認
2. [全体ダンドリ](ダンドリ/) でタスク全体像を把握
3. サブチームの立ち上げ・メンバーアサイン

### サブチームリーダーの方へ

1. 担当チームのダンドリファイルを確認
   * [感門団](ダンドリ/感門団.md)
   * [感門エディスト](ダンドリ/感門エディスト.md)
   * [MCラウンジ](ダンドリ/MCラウンジ.md)
   * など
2. [テンプレート](テンプレート/) からキックオフ議事録のひな形を取得
3. [ナレッジ](ナレッジ/) で過去の知見を参照

### 初めて感門団に参加する方へ

まずこの3つを読んでください：

1. [感門団の役割と心得](ナレッジ/当日運営/mindset.md) - 感門団とは何か
2. [当日ロール別ガイド](ナレッジ/当日運営/roles-guide.md) - 各ロールの動き方
3. [感門団当日ダンドリ](ダンドリ/感門団当日.md) - 当日の流れ

### 初めてエディストに参加する方へ

まずこの3つを読んでください：

1. [感門エディストダンドリ](ダンドリ/感門エディスト.md) - 全体の流れ
2. [JUST記者ガイド](ナレッジ/エディスト/reporter-guide.md) - 記事の書き方
3. [感門エディスト運営](ナレッジ/エディスト/management.md) - チーム運営

### 初めてMCを担当する方へ

まずこの3つを読んでください：

1. [感門之盟の司会とは](ナレッジ/司会MC/about.md) - MCロールの基本
2. [MCとして意識できるあれこれ](ナレッジ/司会MC/tips.md) - 詳細アドバイス
3. [MCラウンジダンドリ](ダンドリ/MCラウンジ.md) - 準備の流れ

### 初めて冊子編集に参加する方へ

まずこの3つを読んでください：

1. [冊子ダンドリ](ダンドリ/冊子.md) - 全体の流れ
2. [冊子制作ダンドリ](ナレッジ/冊子制作/process.md) - 制作工程
3. [合同編集のポイント](ナレッジ/冊子制作/joint-editing.md) - 複数講座での編集

### 基本ルール

* BOXにアップしたファイルは必ずBOX内で編集
* ローカル修正時は必ずファイル名を変更してアップ
* 古いファイルは「履歴」フォルダに格納
