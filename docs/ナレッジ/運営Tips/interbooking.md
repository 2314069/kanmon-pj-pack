## 本受付（井寸房）

### 本の受け取り

1. **お名前、メッセージ記入があるか必ず確認**
    - ない場合は、その場でペンを渡して書いてもらうこと。

2. **本と引き換えに応冊札をお渡し**
    - 本交換時に必要なのでなくさないようにとお伝え
    - ※名札の裏などに入れておくとよい旨伝える

### 預かった本のインターブッキング用本準備

（受付しながら～突破式が始まる前あたりで完了）

3. ｓｅｉｇｏｗスタンプを押す ←※今回押すなら
4. カバーをかける

## インターブッキングタイムの案内（本棚劇場）

{% hint style="info" %}
**案内担当**

古谷師範。4-6の休憩前にステージで案内する
{% endhint %}

### 伝えること

- 井寸房で本の交換を行うこと
- 受付で渡した「応冊札」との交換となること
- 選んだ本が、自分の出した本だった場合のみ再度選び直すこと

---

## インターブッキングタイム（4-7の休憩時間：井寸房）

### 流れ

1. 井寸房に来た人から札をもらう
2. 本を1冊選んで持っていってもらう
3. 自分の出した本だった場合は再度選び直し
4. 札を受け取り～本を選んでもらい～選んだ本を確認してもらうを繰り返す

{% hint style="info" %}
**ポイント**

人数多く、時間もないのでどんどん流れ作業で！
{% endhint %}

### 注意事項

{% hint style="warning" %}
**対応が必要なケース**

- 交換が終わらなかった場合は、6-5の休憩でも行う
- 不在だが、本のみの参加（大濱師範・新垣師範他）があった場合はその人たちの分の本を残った本から選ぶ
- 最終的には、余りがないはず
{% endhint %}
