# 感門之盟 ロール定義

> 感門準備に必要なロールと責任範囲を定義したドキュメントです。

---

## メインロール

### 感門準備統括
- **責任**: 各種ロールのアサイン、各種サブチームの進捗確認、情報統括、MTG設定、企画の実行責任
- **人数**: 1名

### 感門団準備
- **感門準備統括**
  - 感門団の実行責任、必要ドキュメントの作成責任
  - 1名
- **講座PL**
  - 講座と感門団の情報連携、ロール設定、ドキュメントの作成
  - 各講座から1名〜数名
- **当期感門団**
  - 当期からの感門団
- **当期外感門団**
  - 当期外からの感門団

### 感門エディスト
- **感門エディスト統括**
  - 記事企画の実行責任、ライターロールのアサイン、プレ・ポスト記事の管理、配信スケジュール管理
  - 1名
- **講座PL**
  - 講座と感門エディストチームの情報連携、ロール設定、ドキュメントの作成
  - 各講座から1名〜数名
- **JUST記者**
  - 当日の当期外のJUST記者。上杉さんと連携する。

### 冊子
- **冊子デスク**
  - 冊子編集の進捗管理、情報管理、メンバーアサイン、企画の実行
  - 1名
- **学林局側カウンターパート**
  - 1名

### MCラウンジ
- **MCラウンジ統括**
  - カマエ作り、次第共有、タスク共有、原稿レビュー、司会のモデル作り
  - 1名
- **総合司会**
  - 感門日ごとに1名~2名
- **コーナー司会（冠界式、出世魚）**
  - コーナーごとに1名~2名

### アフ感
- **汁匠**
  - アフ感場所決定、下見、アフ感の次第作り、案内作り
  - 各講座から1名〜数名

### 次第作成
- 次第の作成、座席表の作成
- 感門日ごとに1名

## 当期・期外の役割分担ガイドライン

### 推奨される体制

| フェーズ | 当期師範 | 期外感門団 |
|---------|---------|-----------|
| 事前準備 | 主導・決め事・制作物 | 参画・学び |
| 当日運営 | 登壇・学衆フォロー | 裏方業務に専念 |

### 期外感門団を入れるべき理由

1. **当期師範の負担軽減**
   - 当期の方々は、感門される側として、また師範代や学衆をフォローする役割に集中
   - 登壇を控えている中でのダブルロールは難しい

2. **継承の観点**
   - 期を超えて、みんなで言祝ぐひとときにする
   - 感門団がこれまで培ってきたことを繋げていく

3. **イシス編集学校への関わり方**
   - 師範陣以外でも編集学校に関われる場
   - 講座ロールは担っていないが関わりたい人の受け皿

### 当期師範が感門団を担う場合の注意

- ロールに徹することが難しくなりがち
- 事前のルル共有が特に重要
- 当日朝のmtgだけでは弱い

### 同朋衆の位置づけ

- 当期であっても感門団として動きやすい
- 講座に閉じない状態で動かせる

## 感門団ロール（当日）

| ロール | 概要 |
|-------|------|
| 統括 | 感門団全体の取りまとめ、タスク割り振り |
| 受付 | 名札配布、参加者チェック |
| **受付待機** | オープン後も常駐、途中参加・退席者対応 |
| タイムキーパー | 時間管理、札出し |
| マイク出し | 登壇者へのマイク受け渡し |
| カウンター | コーヒー・軽食提供 |
| クローク | 荷物預かり |
| 軽食準備 | 軽食の準備・提供 |
| 席移動・誘導係 | 参加者の席案内、移動誘導 |

> **注意**: 「受付待機ロール」は忘れがち。必ず決めておくこと。

## エディストロール

| ロール | 概要 |
|-------|------|
| 統括 | エディストチーム全体の取りまとめ |
| JUST記者ライター | 当日の記事執筆（当期外） |
| 当期講座からのライター | 当日の記事執筆（当期） |
