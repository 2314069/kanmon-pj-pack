# 感門之盟 ツール・準備物リスト

> 感門当日に必要なツール・準備物の一覧とチェックリストです。

---

## 受付まわり

### 必須アイテム

- [ ] 名札ケース
- [ ] ネックストラップ（色ごとに仕分け）
- [ ] 受付名簿（50音順 or 教室ごと）
- [ ] ペン（記入用）
- [ ] お釣りのお金

### 名札関連

- [ ] 印刷済み名札（教室ごとにまとめる）
- [ ] **白紙の名札用紙**（必須！）
  - 名簿に反映できていなかった助っ人用
  - お名前の漢字が違う方用
  - フォーマットは同じで、名前やロール部分が手書きできるもの

### 配布物

- [ ] ペットボトル（お水）
- [ ] ペットボトルに貼る紙（ラベル）
- [ ] タブロイド

## ゴミ処理関連

- [ ] ゴミ袋（45ℓ〜サイズ別）
- [ ] ゴミ分別用シール
- [ ] 生ゴミ用小袋（カウンター内用）

### 分別

| 種類 | 処理方法 |
|-----|---------|
| ペットボトル | 分別袋へ |
| 燃えるゴミ | 分別袋へ |
| 段ボール | まとめて |
| 缶 | 分別袋へ |

> 業務用廃棄物となるため、袋にシールを貼って裏駐車場スペースのゴミタンクへ廃棄
