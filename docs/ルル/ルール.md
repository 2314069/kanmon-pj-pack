# 感門之盟 ルール

> 感門準備で守るべきルール・運用方針をまとめたドキュメントです。

---

## ドキュメント管理ルール

### BOX利用の基本

1. **BOX内で編集**
   - 1度BOXにアップしたファイルは、必ずBOX内で編集する
   - ローカル保存をしない（もしくは、ローカル保存ファイルをアップロードして上書きしない）

2. **ローカル修正時のファイル名変更**
   - BOXにアップしたファイルをローカルで修正する場合は、必ずファイル名を変更してアップ

3. **古いファイルは履歴フォルダへ**
   - ローカルで修正した時の古いファイルは「履歴」フォルダに格納
   - 作業フォルダには最新のもののみを置く

### バージョン管理の改善

> ファイル名や日付でバージョン管理するだけでは不十分

**推奨：コミットメッセージ風の記録**

```
ver1 2025-09-25 たたき台作成 南田
ver2 2025-09-26 遅刻と早退の情報追加 北田
ver3 2025-09-27 遅刻と早退の情報修正 東田
```

**記録する内容：**
- バージョン番号
- 日付
- 変更内容
- 変更者

## コミュニケーションルール

### ラウンジ運用

1. **スレッドタイトルの更新**
   - スレッドのタイトルは適宜、どんな内容について書いているのか変える
   - 話題が発展したら新しいスレッドを立てることも検討

2. **宛先の明記**
   - 誰に向けた投稿かを明確に
   - 全員向け、特定チーム向けなどを区別

3. **ラウンジの使い分け**
   - 「感門●●」というラウンジがいくつかある場合、どの話題がどのラウンジか明確に

### 積極的な情報共有

> 分かっていそうなことでも連絡は常に積極的発信

- 確認できたことは共有する
- 不明点は早めに質問する
- 決定事項は関係者全員に周知

### editcafe利用の注意点

- 700を越えるやりとりになることも
- タイトルや宛先でそれなりに選別できるが、見落としのリスクあり
- **優先して見るべきものを明確に**

## イレギュラー対応ルール

### 事前に決めておくべきこと

> 想定されるイレギュラー対応は事前に決めておき、守ボードで共有。
> 勧学会で質問されたらいちいち確認することなくさっと回答できるようにしておく。

### 決定が必要な項目

| カテゴリ | 決めておくこと |
|---------|---------------|
| 家族同伴 | 同伴可否、参加費、席の確保方法 |
| 遅刻・早退 | 可否、参加費の扱い、連絡方法 |
| 当日キャンセル | 連絡先、席の調整方法、配布物の扱い |
| アフ感キャンセル | キャンセル期限、キャンセル料 |
| 駐輪場 | 自転車を停める場所 |

### 共有場所

- 守ボードで共有
- 感門団にも周知

- [ ] 消火器の位置

- [ ] 誘導方法

## 当日のコミュニケーションルール

### グループLINEの活用

**目的：**
- 連携を柔軟に取れるようにする
- 急な対応（おやつ受け取りなど）の連絡

**運用ルール：**
- 全員参加が望ましい
- バタバタするとLINEする余裕もなくなるので、事前の段取りが重要
- 少人数・当期感門団であればこそ、連携の重要さを意識

## 準備物の管理ルール

### 所在の明確化

> 「どこにある？」を減らすために書いたものを用意する

**よくある「どこにある？」：**
- 名札
- インターブッキングの本の交換札
- 紙コップ、コーヒーフィルター、コーヒーの粉
- ゴミ袋、ゴミ分別用シール
- 掃除道具、トイレットペーパー、ペーパータオル

### 改善提案

- **在処リストを作成**して当日共有
- 初めての人でも分かるように
